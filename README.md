# E-Pharmacy WordPress

Online pharmacy website built with WordPress.

## Technologies

- WordPress
- GeneratePress Child Theme
- Gutenberg
- GenerateBlocks
- WooCommerce
- Advanced Custom Fields PRO
- Custom Post Type UI
- Filter Everything
- Rank Math SEO
- LiteSpeed Cache
- ShortPixel Image Optimizer

## Theme

Custom child theme:

wp-content/themes/generatepress-child

## Installation

1. Install WordPress.
2. Install required plugins.
3. Import database.
4. Activate GeneratePress Child Theme.
5. Configure WooCommerce.

## Database

Database backup is stored separately.

Import SQL file through phpMyAdmin.
